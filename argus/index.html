What is Argus

A remote command center for Claude Code sessions.

Argus pairs your editor with your phone so the AI can keep coding while you're in a meeting, on the train, or making coffee. You see what changed. You decide what ships.

Code agents got faster than humans can watch.

Claude Code can refactor a service, fix tests, and ship a PR in minutes. The bottleneck isn't the model anymore — it's whoever has to babysit the terminal. Argus lifts that out of the laptop.

Approve from anywhere. Lose nothing.

Risky actions — schema drops, force pushes, secret reads — are gated behind a tap. Everything else streams to a dashboard. Every action is logged, signed, and reviewable later.

How it works

Three pieces. One feedback loop.

No agents on your code. No data hoarding. The extension keeps everything local; only the events your phone needs cross the wire.

01

VS Code Extension

Hooks into Claude Code's session events on your machine. Streams plan, edits, tool calls, and approval requests. Code never leaves the IDE.

02

Realtime Channel

Encrypted Supabase Realtime channel keyed to your device pair. Sub-200ms round-trip from terminal → notification → tap → terminal.

03

Mobile App

iOS & Android. Live session feed, diff previews, approve/deny in one tap, push notifications, audit history. Pair via QR — done in 8 seconds.

Core features

Everything you need to stay in the loop.

Designed for the moment your AI ships work without you watching.

Remote approvals

Risky actions pause the session, ping your phone, and resume the moment you tap approve. No more discovering a dropped table on Monday.

Multi-session monitoring

Watch every project running Claude Code at once. Aggregated feed, per-session focus, color-coded status. Switch with a swipe.

Project previews

See diffs, file trees, and edited blocks rendered as native cards on your phone. Token-gated — your code stays local.

Push notifications

Critical events surface as actionable notifications: failed tests, schema changes, secret reads. Quiet hours and project filters built-in.

Secure pairing

Time-bound QR pair, asymmetric keys per device, revocable tokens. Lose your phone — kill the session in one click from the dashboard.

Audit-ready logs

Every action — model, tool, file, decision, latency — captured and signed. Export as JSON, replay sessions, drop into your compliance review.

Security

Built so you can trust the loop.

Argus is a control plane, not a data plane. Your code stays in your editor — only the events your phone needs travel, and they travel encrypted, scoped, and signed.

  • Human approval before risky actions.Schema drops, force pushes, secret reads, sudo, and shell-out are gated by default. You can scope policies per project.
  • Token-gated previews.Diffs and file trees are short-lived, encrypted, and revocable. Phones never receive raw source unless you ask for it.
  • Device pairing with asymmetric keys.QR-based pair using a fresh keypair per device. Revoke any device from the dashboard with one click. Lost phones become inert immediately.
  • Signed session logs.Every action is hashed, signed, and timestamped — replayable for incident review or audit. Export to JSON or stream to your SIEM.
  • Privacy-first architecture.No code uploaded. No model training on your sessions. The relay sees event metadata, not content. Self-hosted relay coming for Pro.
