# FZ Labs — fzlabs.com

Marketing landing page for FZ Labs, an AI-native software company building Argus, SignalNews, Swarmix, and F0 IDE.

## Stack
- Static HTML / CSS / vanilla JS — no build step.
- Hosted on Vercel.

## Files
- `index.html` — main landing page
- `signalnews-apps.html` — SignalNews mobile apps "Coming Soon" page
- `logos/` — brand and product imagery
- `vercel.json` — clean URLs + cache headers
